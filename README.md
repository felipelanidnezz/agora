# ÁGORA - Centro de Desarrollo y Estimulación Temprana

SPA construida con Quasar Framework y Vue 3 para presentar la propuesta institucional de ÁGORA.

## Requisitos

- Node.js 18+
- npm 9+

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

La aplicación se levanta en modo desarrollo con `quasar dev` y soporta recarga en caliente.

## Linter

```bash
npm run lint
```

## Compilación para producción

```bash
npm run build
```

Los artefactos quedan en la carpeta `dist/`.

## Funcionalidades destacadas

- Navegación con scroll suave a secciones ancladas
- Secciones modulares para misión, niveles, jornadas, instalaciones, actividades, equipo y contacto
- Carruseles táctiles (`q-carousel`) para mostrar instalaciones y eventos
- Formulario de contacto con validación y notificaciones de Quasar
- Botón flotante de WhatsApp siempre visible (`q-fab`)
- Animaciones sutiles al hacer scroll y diseño responsive mobile-first

## Placeholders

Las imágenes utilizan `picsum.photos` y enlaces de redes/mapas de ejemplo. Sustituir por los recursos oficiales antes del despliegue.
